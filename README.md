# BRCA1 FASTA Analysis

## Overview
This project analyzes the BRCA1 gene sequnece obtained from NCBI database in FASTA format.

### Objective 
- Read a FASTA file using BioPython
- Extract the DNA sequence
- Calculate sequence length
- Count nucleotides (A, T, G and C)
- Calculate GC content
- Generates complement DNA sequence
- Translates the DNA sequence to protein

## Dataset
Source: NCBI Gene Database
Gene: BRCA1 (Breast Cancer 1)
Organism: Homo sapiens
File format: FASTA

## Tools Used
- Python
- Biopython
- Visual Studio Code

## Output 
The program displays:
- Sequence ID
- Sequence length
- Number of A, T, G, C nucleotide
- GC content
- Complement DNA sequence
- Proteins
